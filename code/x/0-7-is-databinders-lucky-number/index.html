0.7 is Databinder's lucky number

By n8han, on August 21, 2006 { databinder, wicket }

Databinder 0.7 is out, with DataView support built-in. This release has all the ingredients of a basic data-driven webapp toolkit.

We’ll let this one run around in the wild for a few weeks, fix any problems that come up, and then skip straight to 1.0. Unless someone has a terrible need for some medium-sized feature…. a side of mashed potatoes?

There’s no rush, except that I’d like for Databinder 1.0 to be out for a little while before people start shifting their interest away from the Wicket 1.2 branch.

What, there’s something besides the 1.2 branch? Shh!
