Databinder 0.2

By n8han, on February 7, 2006 { databinder, wicket }

Released a new version of Databinder today. Its improvements include basic transaction handling, a more flexible session factory, and some handy components and validators.

I also worked on the example applications, streamlining the Phone directory and adding a new Bookmarks example.

Special thanks to T. Bennett and J. Brown for their comments and suggestions.
